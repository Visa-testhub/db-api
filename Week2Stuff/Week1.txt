This is SQL queries from pet table part 1
Pet-taulun tekeminen on esitetty materiaalin osassa 2. Syötä ensin kaikkien esimerkissä annettujen lemmikkien tiedot tauluun. Arvioitavat tehtävät:

a) Poista Puffballin tiedot taulusta.
b) Muuta Bowserin syntymäajaksi 31.8.1998
c) Chirpy on kuollut 15.9.2000. Lisää tämä tieto tauluun.
d) Tulosta kaikki lemmikit, jotka ovat syntyneet 1990-luvulla. (Tulosta kaikki tietueen tiedot)
e) Tulosta kaikki miespuoliset koirat. (Tulosta kaikki tietueen tiedot)
f) Tulosta ainoastaan kaikkien miespuolisten koirien nimet.
g) Tulosta kaikkien miespuolisten koirien nimi ja omistajan nimi.

these were the questions, and here are my answers to them:
a) DELETE FROM pet WHERE name = 'Puffball';
b) UPDATE pet SET birth = '1998-08-31' WHERE name = 'Bowser';
c) UPDATE pet SET death = '2000-09-15' WHERE name = 'Chirpy';
d) SELECT * FROM pet WHERE birth LIKE '199%';
e) SELECT * FROM pet WHERE species='dog' AND sex='m';
f) SELECT name FROM pet WHERE species='dog' AND sex='m';
g) SELECT name,owner FROM pet WHERE species='dog' AND sex='m';
